# AR Chicken Project

## Description
This project showcases a 3D chicken model with AR integration for mobile and tablet devices. Users can view and interact with the 3D models in augmented reality and explore the website’s content.

## Live Site
[View Live Site](https://your-live-site-link.com)

## How to Use AR
1. Open the site on a mobile or tablet browser.
2. Click the "View in AR" button on the model.
3. Follow the on-screen instructions to place the 3D model in your environment.

## Project Structure
- `index.html` → main HTML file
- `style.css` → CSS styles
- `script.js` → JavaScript functionality
- `models/` → folder containing 3D models
- `video_screen_rec/vid.mp4` → demonstration video
- `video_instructions.txt` → instructions for the demo video

## Features
- Interactive 3D models
- AR functionality for mobile and tablet
- Responsive design
